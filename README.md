## git command

|  git 命令  |  說明  |
| ------ | ------ |
|  git pull | 拉下檔案至本地 |
| git push | 推上檔案至遠端 |
| git add \<file> | 推至暫存區 |
| git add . | 全部推至暫存區 |
| git commit --m "內容" | 推進本地端 |
| git checkout \<branch> | 切換分支 |
| git merge \<branch> | 合併分支 |
| git status | 顯示修改檔案清單 |
| git log | 顯示提交記錄 |
